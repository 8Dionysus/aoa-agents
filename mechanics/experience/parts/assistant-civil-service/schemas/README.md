# Assistant Civil Service Schemas

These schemas define assistant civil/service companion contracts. They are
agent-layer contracts for service posture and reader examples, not live support
automation, runtime policy, or Agon arena authority.

| Schema | Contract |
| --- | --- |
| [agent-governance-posture.schema.json](agent-governance-posture.schema.json) | reviewed governance posture |
| [assistant-variant.schema.json](assistant-variant.schema.json) | assistant kind adjunct |
| [service-identity.schema.json](service-identity.schema.json) | service identity adjunct |
| [service-contract.schema.json](service-contract.schema.json) | service contract adjunct |
| [service-governance.schema.json](service-governance.schema.json) | governance and rollback adjunct |
| [service-certification.schema.json](service-certification.schema.json) | certification posture adjunct |
| [civil-formation.schema.json](civil-formation.schema.json) | public reader-path example contract |

Arena exclusion lives in the adjacent
[arena-exclusion schema route](../../arena-exclusion/schemas/arena-exclusion.schema.json).
