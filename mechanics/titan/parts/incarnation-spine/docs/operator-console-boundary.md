# Titan Operator Console Boundary

The Titan operator console is a visible control surface for the first five Titans. It is not a role-authoring surface and not an autonomous background runtime.

| Titan | Role | Lane | Default | Gate |
|---|---|---|---:|---|
| Atlas | architect | structure | yes | none |
| Sentinel | reviewer | risk | yes | none |
| Mneme | memory-keeper | memory | yes | none |
| Forge | coder | implementation | no | mutation |
| Delta | evaluator | verdict | no | judgment |

Boundary law: the console may display roster, record events, record approvals, and generate app-server launch plans. It may not silently spawn subagents, silently mutate files, or override source-owned role truth.
