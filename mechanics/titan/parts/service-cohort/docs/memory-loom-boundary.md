# Titan Memory Loom Boundary

The Memory Loom is a service-cohort continuity layer. It helps the operator remember what happened, where a claim came from, and which owner surface should be checked next.

## Boundary law

Mneme may weave candidate recall. Mneme may not canonize recall.

## Stronger than Memory Loom

1. Live operator instruction.
2. Owner-repo source file.
3. Wave manifest and source seed.
4. Reviewed receipt or planting report.
5. Memory Loom digest or recall result.

## Titan duties

- Atlas: annotate structural owner boundaries.
- Sentinel: mark drift risks and authority conflicts.
- Mneme: form recall candidates with provenance.
- Forge: no memory mutation without explicit mutation gate.
- Delta: no verdict from recall without explicit judgment gate.

## Forbidden shapes

- Hidden memory authority.
- Silent personal-memory writeback.
- Deleting or masking records without a tombstone.
- Turning recall confidence into proof.
