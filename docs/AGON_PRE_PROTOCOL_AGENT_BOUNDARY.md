# Agon Pre-Protocol Agent Boundary

`aoa-agents` owns actor-facing formation. It does not own Agon protocol law.

After Waves I, II, and II.5, this repository may say:

- which current role homes have agonic forms
- which current role homes have assistant variants
- which seats they may later become candidates for
- which seats remain forbidden
- which surfaces still require center-owned charter before becoming live

It still must not define:

- arena sessions
- sealed commits
- lawful moves
- challenge packets
- contradiction ledgers
- verdict logic
- scar storage
- retention checks
- ToS promotion

## Distinctions

Eligibility is not authority.

Candidate seat is not live seat.

Witness candidacy is not scar ownership.

Judge candidacy is not verdict sovereignty.

Assistant escalation is not hidden contestant status.
