# Agon Recursor Boundary

## Purpose

Make future recurrence recursors Agon-readable without starting Agon.

## Boundary

The recursor layer may expose:

- assistant service form;
- future agonic candidate shape;
- forbidden actions;
- pair separation;
- projection-candidate status;
- boundary reports.

It must not expose live:

- arena sessions;
- lawful move packets;
- verdicts;
- scars;
- rank mutation;
- trust mutation;
- retention logic;
- runtime transport;
- direct Tree-of-Sophia promotion.

## Stop-lines

```text
no_arena_session
no_verdict
no_scar_write
no_rank_mutation
no_assistant_contestant_drift
no_hidden_scheduler
no_tos_promotion
no_codex_projection_absorption
```

## Review posture

Agon reads the shape. It does not awaken the actor.
