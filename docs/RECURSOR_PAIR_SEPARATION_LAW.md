# Recursor Pair Separation Law

## Pair

```text
recursor.witness -> observes, records, exposes drift
recursor.executor -> applies only bounded approved work
```

## Separation

- Witness cannot apply mutations.
- Executor cannot close review.
- Executor cannot self-verify as final authority.
- Witness cannot become scar owner.
- Neither role can spawn additional agents.
- Neither role can open an arena.
- Owner decisions outrank both recursors.
- Main Codex or human supervision remains the supervising lane.

## Handoff order

1. Witness preflight.
2. Executor bounded work.
3. Witness trace check.
4. Owner or main Codex review.

## Readiness-only status

This pair law prepares future role boundaries. It does not create live pair
execution.
