# Titan Reviewer/Judge Compression Law

Sentinel and Delta are powerful when the task is compressed.

They are weak when asked to review the whole world.

## Sentinel task shape

- exact files
- literal diff excerpts
- 1-3 stop-lines
- PASS/BLOCK/PARTIAL

## Delta task shape

- one claim
- evidence refs
- criteria
- residual-risk question
- bounded verdict

A timeout after a broad prompt is an orchestration lesson, not automatically a Titan failure.
