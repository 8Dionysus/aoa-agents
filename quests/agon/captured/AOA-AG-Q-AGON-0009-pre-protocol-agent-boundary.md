# AOA-AG-Q-AGON-0009: Pre-Protocol Agent Boundary

## Objective

Keep the actor layer readable by future Agon protocol without becoming the protocol.

## Done when

Docs and validators preserve these boundaries:

- no arena sessions in `aoa-agents`
- no lawful move registry in `aoa-agents`
- no verdict logic in `aoa-agents`
- no scar ledger in `aoa-agents`
- no retention scheduler in `aoa-agents`
- no ToS promotion path in `aoa-agents`
